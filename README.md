# 🎾 Bouncing Ball - Color Changing

A fun and simple bouncing ball animation built in C# Windows Forms. Perfect for TikTok, YouTube Shorts, and Instagram Reels content creation.

## ✨ Features

- Smooth 60 FPS animation
- Ball bounces realistically off all four walls
- **Changes color on every wall hit** (random bright colors)
- Nice shadow and highlight effects
- Clean and simple interface
- Easy to record for vertical videos

## 🚀 How to Run

1. Open the project in **Visual Studio** (2022 recommended)
2. Press `F5` to start
3. The ball will start bouncing and changing colors on impact
4. Press `Esc` to close the application

## 🎨 Customization

You can easily change:

- Ball size (`ballSize`)
- Speed (`dx` and `dy`)
- Window size (`this.Size`)
- Initial color
